# vimsay

## Install

```vim
NeoBundle '844196/vimsay', {
    \ 'depends': 'Shougo/unite.vim'
    \ }
let g:vimsay_cowpath = '/usr/local/share/clangsay/cows'
```

## Usage

```vim
:Vimsay renge.cow
```
